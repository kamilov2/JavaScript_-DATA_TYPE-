JavaScript ma'lumot turlari bo'yicha qo'llanma

1. String

Matnni ifodalash uchun ishlatiladi.

        
let str = "Salom, Dunyo!";
console.log(str);

2. Number

Sondan iborat qiymatlarni ifodalash uchun ishlatiladi. Butun sonlar va o'nli sonlar.

        
let num = 42;
let pi = 3.14;
console.log(num);
console.log(pi);

3. Boolean

Ikkilik qiymatlarni ifodalash uchun ishlatiladi: true yoki false.

        
let isTrue = true;
let isFalse = false;
console.log(isTrue);
console.log(isFalse);

4. Null

Hech qanday qiymatni ifodalash uchun ishlatiladi.

        
let emptyValue = null;
console.log(emptyValue);

5. Undefined

Qiymati belgilanmagan o'zgaruvchini ifodalash uchun ishlatiladi.

        
let undefinedValue;
console.log(undefinedValue);

6. Object

Murakkab ma'lumot tuzilmalarini saqlash uchun ishlatiladi. Ob'ektlar xususiyat va metodlardan iborat.

        
let person = {
    name: "Ali",
    age: 25,
    isStudent: true
};
console.log(person);

7. Array

Bir xil turdagi ma'lumotlarni tartibli ro'yxat sifatida saqlash uchun ishlatiladi.

        
let fruits = ["Olma", "Banan", "Gilos"];
console.log(fruits);

8. Function

Qayta ishlatiladigan kod bloklarini yaratish uchun ishlatiladi.

        
function salomBer(name) {
    return "Salom, " + name + "!";
}
console.log(salomBer("Dunyo"));
